# 模链云（ModLinkCloud）产品说明书

**文档版本**：1.0  
**更新日期**：2026-05-10  
**产品中文名称**：模链云  
**产品英文名称**：ModLinkCloud  

---

## 1. 文档说明

本说明书用于向内部团队、合作伙伴或潜在客户说明 **模链云** 的产品定位、能力范围与交付形态，不构成法律承诺；具体权利义务以正式合同及届时有效的 **《用户协议》**、**《隐私政策》** 为准。

---

## 2. 产品概述

**模链云** 是一款面向企业与开发者的 **智能化 SaaS 产品**，在统一账号、权限与计费体系下，为用户提供对话、自动化与集成等能力。产品的 AI 推理能力由 **经授权的第三方模型服务**（例如通过聚合平台接入的多厂商大模型）在后台提供；模链云聚焦于 **产品体验、用量治理、安全合规与可运营性**，而非向终端用户裸售上游 API 目录。

---

## 3. 定位与价值主张

| 维度 | 说明 |
|------|------|
| **统一入口** | 用户在本产品中完成登录、配额查看、用量与账单自助服务，降低多厂商对接复杂度。 |
| **可运营** | 提供管理侧配置、订单与充值、模型与渠道策略等能力，支撑商业化与规模化运营。 |
| **可治理** | 支持限流、额度、黑白名单与调用监控，降低滥用与资金风险。 |
| **可观测** | 提供用量统计、报表与日志查询能力，便于成本分析与问题追溯。 |

---

## 4. 目标用户

- **终端用户**：使用模链云所提供的应用能力完成业务目标的企业成员或个人用户（具体以实际开放注册策略为准）。  
- **管理员 / 运营人员**：在管理后台进行用户、渠道、模型、订单与系统配置管理。  
- **开发者**：在授权范围内使用 **API / 集成能力**（若产品提供），用于与企业内部系统或自动化流程对接。

---

## 5. 功能范围（与产品规划对齐）

以下模块描述 **规划中的能力边界**，随版本迭代可能增删；以实际上线与 release notes 为准。

### 5.1 管理员后台

- 用户与权限管理  
- 上游渠道与路由策略管理  
- 模型目录与可用性配置  
- 订单、充值与财务相关操作入口  
- 系统级参数与安全策略配置  

### 5.2 用户前台（自助门户）

- 账号注册 / 登录与基础资料  
- API 密钥管理（若对该用户角色开放）  
- 余额或额度充值与查询  
- 用量与账单自助查询  
- 调用说明与帮助文档（含集成示例）  

### 5.3 核心网关能力（对用户透明、对运维可配置）

- 将业务侧请求 **路由** 至经配置的上游推理服务  
- **流式与非流式** 响应支持  
- 请求参数在合规与安全策略下的 **透传与校验**  
- 与计费、日志、风控联动的基础能力  

### 5.4 计费与财务

- 计费规则配置（例如按用量档位、包月包量等，以实际上线规则为准）  
- 与额度联动的 **实时扣减**（或预授权 + 结算，以实现为准）  
- 账单与明细查询  

### 5.5 风控与治理

- 限流与配额  
- 黑白名单  
- 调用频率与异常监控  

### 5.6 数据统计与报表

- 多维度用量统计（如按用户、应用、模型、时间等，以实现为准）  
- 管理看板与关键指标展示  
- 调用与审计类日志查询能力  

### 5.7 安全与运维

- 传输与存储安全基线（随实现细化）  
- 监控与告警  
- 运维与发布相关内部能力（是否对客户可见以实际产品为准）

## 6. 技术架构说明（仓库划分）

为便于研发协作与发布，工程拟拆分为多个代码仓库，典型划分如下（名称与边界以仓库 README 为准）：

| 仓库 | 职责概要 |
|------|-----------|
| **modlink-cloud** | 总仓库：文档、约定、编排或与子仓库的协作说明 |
| **modlink-gateway** | 后端接口与网关转发服务 |
| **modlink-cloud-admin** | 管理后台前端 |
| **modlink-cloud-web** | 用户端前台 |

## 7. 第三方依赖与免责声明（摘要）

- **模型与推理服务** 由第三方提供，其 **可用性、延迟、输出质量、适用场景及定价** 均可能变更；模链云将尽力通过架构与运维手段保障服务连续性，但 **不对上游服务的持续可用性或某一模型的永久提供作出保证**。  
- 用户通过模链云访问 AI 能力时，须遵守 **模链云用户协议** 以及 **相应模型提供方** 公布的条款与政策（将在用户协议中通过援引或摘要方式提示）。  
- 若产品提供对外 API，该 API 属于 **模链云服务的一部分**，用于集成与自动化；其权限、模型范围与配额以账户配置及产品策略为准。
